เพิ่มเพจ Facebook (รองรับหลาย Token)

วาง Access Token ได้ทั้งแบบ User token (ระบบจะแสดงรายชื่อเพจให้เลือก) หรือ Page token (เชื่อมต่อทันที) — ต้องมีสิทธิ์ pages_show_list และ pages_messaging. เพิ่ม Token ได้หลายชุด แต่ละชุดจะดึงเพจของตัวเองเข้ามารวมกัน

เลือกเพจที่ต้องการดึง inbox
วิธีหา Page Access Token
  1. เข้า Graph API Explorer
  2. เลือกแอปของคุณ แล้วเลือก Get Page Access Token จากเมนู User or Page
  3. เพิ่มสิทธิ์ pages_messaging, pages_read_engagement, pages_show_list
  4. เลือกเพจที่ต้องการ แล้วคัดลอก token มาวางที่นี่
